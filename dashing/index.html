What is Dashing

Dashing is a Sinatra based framework that lets you build beautiful dashboards.

Key features:

Premade widgets easily distributable using gists

Easily create your own with scss, html, and coffeescript

Widgets have a unified updating mechanism that keep things DRY and simple (batman.js)

Simple API to push data to your dashboards

Dashboards can be hosted on Heroku extremely easily

Why do we use it in Tripsta

We aim to create a multi-dashboard application, integrating dashboards from Newrelic, Kibana, Graphite, Munin, Jenkins and other performance monitor tools.

Can include visual or other indicators for issues.

Can be used it team TVs and have additional team specific widgets (YouTube broadcasting, countdowns, shoutboxes etc).

Anatomy of a widget

- an HTML file used for layout and bindings

- a SCSS file for styles

- a coffeescript file which allows you to handle incoming data & functionality

Dashboard is a collection of widgets.

Dashing uses rufus-scheduler to schedule jobs.

Post the data you want in json to /widgets/widget_id
